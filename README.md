# 씨미 챗봇 (cime_chat_bot)

씨미(ci.me) 스트리밍 플랫폼 전용 챗봇입니다.
채팅 명령어 자동응답, 출석 체크, 후원 감사 메시지, 룰렛 시스템을 지원하며, 웹 대시보드로 손쉽게 관리할 수 있습니다.

---

## 주요 기능

- **채팅 명령어** — DB에 저장된 명령어에 자동 응답 (변수 치환 지원)
- **출석 체크** — 매일 오전 5시 초기화, 누적 출석 횟수 관리
- **후원 감사** — 후원 발생 시 자동 감사 메시지 전송
- **룰렛 시스템** — 후원 트리거로 룰렛 자동 실행, OBS 오버레이 연동
- **웹 대시보드** — 명령어·룰렛·설정 관리, 기록 조회 및 CSV 추출

---

## 사전 준비

- Python 3.10 이상
- Google Chrome + ChromeDriver (Selenium 자동 로그인에 사용)

---

## 슬러그 (Streamer Slug)

슬러그는 씨미 방송 주소에서 `@` 뒤에 오는 고유 ID입니다.

예: 방송 주소가 `https://ci.me/@so0420` 이면 슬러그는 `so0420`

봇 설정 화면의 **스트리머 슬러그** 항목에 입력하면 됩니다.

---

## 설치 및 실행

```bash
pip install -r requirements.txt
python cime_bot.py
```

실행 후 브라우저에서 http://localhost:8000 에 접속하면 관리 대시보드가 열립니다.

---

## Windows exe 사용 (빌드 없이 바로 실행)

릴리즈된 `cime_bot.exe`를 원하는 폴더에 놓고 실행하면 됩니다.
`bot.db`와 `web/` 폴더는 exe 옆에 자동으로 생성됩니다.

---

## 빌드 (직접 빌드하려는 경우)

```bash
pyinstaller --noconfirm --onefile --icon "favicon.ico" --add-data "web;web" --collect-all selenium cime_bot.py
```

빌드 결과물은 `dist/cime_bot.exe`에 생성됩니다.

## 명령어 변수

| 변수 | 설명 |
|------|------|
| `<보낸사람>` | 채팅을 보낸 시청자 닉네임 |
| `<출석횟수>` | 해당 시청자의 누적 출석 횟수 |
| `<업타임>` | 방송 시작 후 경과 시간 |
| `<방제>` | 현재 방송 제목 |
| `<카테고리>` | 현재 방송 카테고리 |
| `<팔로우>` | 팔로워 수 |
| `<받은금액>` | 후원 금액 |

## OBS 룰렛 오버레이 설정

1. 룰렛 설정 페이지에서 **오버레이 URL** 복사
2. OBS → 브라우저 소스 추가 → URL 붙여넣기
3. 봇이 실행되면 후원 발생 시 자동으로 오버레이에 룰렛이 표시됩니다

> 봇이 꺼진 상태에서 OBS를 먼저 켜도 괜찮습니다. 봇이 켜지면 자동으로 재연결됩니다.
